# Test checklist

Kjør lokalt etter pull:

```powershell
npm run build
npm run dev
```

Kontroller:

- Forsiden fungerer fortsatt.
- Tjenestekortene lenker til tjenestesider.
- Tjenestesidene viser relaterte Kjekt å vite-artikler.
- `/kjekt-a-vite` viser mange artikler gruppert etter kategori.
- Artikkelsidene viser relatert innhold.

Denne filen kan slettes senere dersom ønskelig.
